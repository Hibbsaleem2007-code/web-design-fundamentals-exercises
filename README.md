# HTML Website Project

This project contains multiple HTML webpages created using semantic HTML and basic CSS styling.

## Pages Included

1. Cooking with Andy
A recipe webpage that shows how to bake chocolate chip cookies.  
It uses semantic tags like header, main, article, section, aside, and footer.

2. Our Story
An informational webpage about a fictional company called Machine Gym.  
The page is organized using different sections to explain the company's idea and future.

3. J's Gallery
A gallery webpage that displays images and a video.  
It uses CSS layout techniques such as Flexbox and Grid.

4. The Wondering Photographer
A photography webpage that shows photos with captions and an “Add to basket” button.

## Technologies Used

- HTML5
- CSS3

## Purpose of the Project

The purpose of this project is to practice:
- Semantic HTML structure
- Webpage layout design
- Image and video embedding
- Basic CSS styling
